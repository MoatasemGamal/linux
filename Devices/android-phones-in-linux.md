# Android phones

## Mount android phones in linux

> the are too many ways, I will use simple-mtpfs

### install simple-mtpfs

- fedora linux

  `sudo dnf copr enable vectorsigma/simple-mtpfs`

  `sudo dnf install simple-mtpfs`
